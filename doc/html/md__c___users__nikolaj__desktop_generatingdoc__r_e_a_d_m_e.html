Task Planner

The Task Planner program is developed as an attempt to help consumers schedule their electrical tasks in an efficient manner, in order to mitigate their climate impact. This library contains all documentation available on the program.

GitHub repository

All source code is available in the upgraded-potato repository on GitHub. Link: https://github.com/mirakst/upgraded-potato

Authors
