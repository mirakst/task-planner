Detailed Description

The task structure. Contains all information needed for a task.

Parameters
typeEnum that determines if the task is Active, Passive or Timed. (0, 1 & 2).
nameString that contains the name of the task.
powerDouble that contains the power of the task.
kWhDouble that contains the power consumption of the task.
durationDouble that contains the amount of time the task takes to complete.
start_hrInt for the start time of the task.
end_hrInt for the end time of the task.
days[]An array of the days in a week (there are 7).
total_days_yrInt for calculating the amount of days task is performed throughout the year.
min_valueDouble for the minimum value of completing the task.
max_valueDouble for the maximum value of completing the task.
